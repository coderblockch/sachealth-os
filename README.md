# 🏥 SacHealth OS - Intelligent Triage System
**Combatting hospital saturation in Sacramento, CA.**

## 🚀 Project Overview
SacHealth OS is a HealthTech prototype designed to optimize the flow of critical patients. It uses a 
clinical triage engine to prioritize high-risk patients and a resource management dashboard to reduce 
bottlenecks.

## 🛠️ Tech Stack
- Frontend: HTML5, CSS3 (Dark Mode UI), JavaScript (ES6+).
- Architecture: Graceful Degradation & Client-side Logic.

## 🌟 Key Features
- **AI Triage Engine:** Real-time priority assignment.
- **Bed Management:** Hospital capacity visualization.
- **Insurance Escalation:** (New) Streamlined approval process.
